# Fun with flags / Funny flags

Ang repositoryong ito ay magagamit sa pag-gawa ng laro ukol sa mga bandilang inpisrasyon mula sa palabas na Big Bang Theory, kung saan ang manlalaro ay manghuhula tungkol sa impormasiyon ng isang lugar sa pamamagitan ng bandila nito.

Para dito gagamitin ang hacktoberfest para makalikom impormasyon tungkol sa mga bansa, mayroon nang nangungunang seleksyon ng bandila na maaring mahanap sa path ng imahe:

[Imahe ng Bandila](https://github.com/xaca/juego_banderas/tree/master/img)

Kami ay naglilikom ng kompletong json file na nagtataglay ng mga sumusunod sa bawat bansa, halimbawa:

+ Pangalan
+ Kapital
+ Lengwahe
+ Path ng imahe (gamiting reference nag nasa img na folder)

## Instruksiyon sa pagsali

+ Balikan ang listahan ng mga bansa sa folder na img
+ Update ng kumpleto ang impormasyon ng bawat bansa sa loob ng json file
+ Gumawa ng hindi bababa sa 5 contibusiyon sa archive ng impormasyon ng bansa
+ Sumunod sa mga rules ng event code of conduct
